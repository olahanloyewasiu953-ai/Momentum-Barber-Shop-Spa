# Momentum Barbing Salon & Spa Website

A modern and responsive website designed for Momentum Barbing Salon & Spa.  
The website showcases professional grooming services, hairstyles, spa treatments, and provides an elegant online presence for customers.

## Features

- Modern and clean design
- Responsive layout for mobile and desktop
- Services section
- Image gallery
- Video showcase
- Contact section
- Smooth user experience

## Services

- Professional Haircuts
- Beard Grooming
- Hair Styling
- Pedicure Treatments
- Spa Services

## Project Purpose

This project was created to help Momentum Barbing Salon & Spa establish a professional online presence and allow customers to explore their services easily.
